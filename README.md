# Modelo

https://github.com/unq-ui/instagram-model-js

siguiendo la API provista por la cátedra de Instagram, tuvimos que;

En el TP 1, el de API, conectar nuestro front end con la API.

En el TP 2, el de Web, realizar la página web de Instagram, además de las rutas para que el usuario pueda 
navegar por las páginas indicadas en el Figma provisto por la cátedra, luego, que el usuario
pueda logearse, realizar un post, un comentario, buscar a otro usuario, seguirlo, 
darle like a un post, junto con sus medidas de seguridad

En el TP 3, Mobile, es realizar exactamente lo mismo que el TP 2, pero en versión Mobile

Las tecnologías utilizadas fueron:

API REST: Diseño, consumo y documentación con Swagger.
JavaScript (ES6+), Node.js.

Web & Mobile: HTML5, CSS3, nociones de Next.js y React Native (Expo) para integración de endpoints.
